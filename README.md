# To-Do List Backend

This is a **TypeScript + Express** backend for a simple To-Do List application using **MongoDB**.  
It supports CRUD operations for tasks and includes unit tests using **Jest** and **Supertest**.

---

## Features

- Create, read tasks  
- Input validation (title required)  
- TypeScript type safety  
- Mocked unit tests with Jest  
- MongoDB connection via Mongoose  

---

## Technologies

- Node.js / TypeScript  
- Express.js  
- MongoDB / Mongoose  
- Jest + Supertest  
- dotenv for environment variables
